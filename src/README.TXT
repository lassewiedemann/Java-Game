Traveler-Projekt - BENUTZERHINWEIS:
  
Aus lizenzrechtlichen Gründen kann im Download-Material für die Game Engine die Javazoom-Bibliothek zur Wiedergabe von MP3-Dateien nicht angeboten werden. Sie wird im Rahmen des Projekts nicht benötigt. Sollten Sie das Projekt um Sound zu ergänzen wollen, laden Sie sich bitte von der Engine-Alpha-Website eine aktuelles Standardprojekt der Engine Alpha herunter und kopieren Sie die speziell für das Projekt entwickelten Klassen dort hinein.
